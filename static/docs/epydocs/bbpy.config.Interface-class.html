Class Interface

source code

object --+
         |
        Interface

The Interface class gives access to the locations of a project.

In virtually every case, an anonymous instance may be used. The only disadvantage would be the overhead of re-reading the interface file on every instantiation.

**Note: The bbw.config file in this is only used to find the INTERFACE file.

Example:

>>> from bbpy import config
>>> path = config.Interface().data
