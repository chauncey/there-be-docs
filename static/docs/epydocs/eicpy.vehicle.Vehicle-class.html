getInfoFromVin(self, vin, year, make=None)

source code  
Vehicle.getInfoFromVin('VINHERE', 1993) => {'status': 'GOOD', 'year': 1993, 'make': 'HOND', 'model'  : 'ACCORD EX SEDAN 4D2.2l 4cyl', 'symbol': '12'}

@param vin: A vehicle identification number
@param year: a 2 or 4 digit year
@param make: 4 character code

@returns: a structure containing vehicle information if the
    status is GOOD or partial information if the status is INC.
    Example:
    {'status' : 'GOOD' ('GOOD'|'BAD'|'INC')
    ,'year'   : 1993   (YYYY)
    ,'make'   : 'HOND' ('XXXX')
    ,'model'  : 'ACCORD EX SEDAN 4D2.2l 4cyl'
    ,'symbol' : '12' ('XX') or '12/15' ('XX/XX')
         1st part is the collision symbol and the 2nd part is the comp symbol if present
    }
