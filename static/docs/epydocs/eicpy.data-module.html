getSusVehicles(key)

source code 

Get a list of vehicle records from the BBx Suspense data file.

Parameters:
  • key - And record key without the record code. (example: (0907601080903 BEAR")
Returns:
A list of BBPyStrings containing each vehicle record. If a NONOWNER record exists, and is the only vehicle, an empty list is returned.
Raises:
  • BBPyKeyNotFoundError - BBPyKeyNotFoundError will be raised in the event that no vehicle records are found. (Even NONOWN policies should have 1 record. This is to preserve good record integrity since many BBx programs expect at least 1 record.)

getAgentRecs(agent)

source code 

Lookup the ADP01 records based on the 4 digit agent code.

>>> recs = getAgentRecs('5071')
>>> recs.agent
u'5071'
>>> recs.state
u'OK'
>>> recs.sold_to
u'0000'
>>>
Parameters:
  • agent - 4-digit agent code
Raises:
  • BBPyKeyNotFoundError - BBPyKeyNotFoundError will be raised if no agent exists for the supplied agent number.

pyToJSON(obj, sanitize=True, *args, **kwargs)

source code 

A wrapper function around whichever support module we're using for JSONization. The purpose of this is fix some of the quirks that come up when you mix BBx strings, Python strings, JSON and Page Templates. We should always use this function instead of calling a support module (i.e. simplejson) directly.

Parameters:
  • obj - The python object to convert to JSON
  • sanitize - Specify whether or not to sanitize the output of the Python object -> JSON string conversion. Generally, you want this.
  • args - Value based arguments to be passed to the JSONizing support module
  • args - Keyword based arguments to be passed to the JSONizing support module
Returns:
A python string containing the string version of a JSON object.

rateValuePadNum(instr, padsize)

source code 

This converts "25/50/25" into "025/050/025" or vice versa.

Parameters:
  • instr - the string to convert
  • padsize - how wide the numbers should be padded to. Set to 0 to remove padding. Set to a positive number to place padding.
Returns:
The padding-modified string. If the string contains no numbers, or contains no slashes and is longer than the padsize, then this will have no effect.
