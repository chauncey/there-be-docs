getTpl(templates)

source code 

Convenience function to build a BBx String template from any number of template files.

Parameters:
  • templates - A list of either filenames, file-like objects or BBPyStringTemplates of BBx Templates from which to build a string template. The None object is a valid type to include in the list and will be ignored.
Returns:
A BBPyStringTemplate object built from the templates in the supplied filenames.

getRecords(tpl, datafile, keys=[''], exact=False, keynum=None, numerics=False)

source code 

Convenience function to get a dictionary of BBPyString records from a BBx datafile.

Parameters:
  • tpl - a BBPyStringTemplate object of the template for the record(s). Use getTpl for this.
  • datafile - a filename or file-like object from which to read the records
  • keys - Keys can be a number of things. If you pass None or nothing at all, getRecords will attempt to apply the template to all records in the supplied datafile. If you pass a list of strings containing keys, getRecords will retrieve all records for matching keys. The default value (with exact as False) will return all keys in a datafile.
  • exact - Determines if the passed keys should be matched exactly or if getRecords should return partial keys as well.
  • keynum - See keynum for mkeyed.MKEYEDReader.read
  • numerics - Make records returned a tuple of (record, [list of numerics])
Returns:
A dictionary of BBPyString objects containing a list of data records with the BBx string template applied (as a BBPyStringTemplate object). The dictionary key will be the key value passed. If no key is specified then only a list will be returned. If exact is True, then the dictionary will only contain the value and not a list of values. Example: 
   exact = False (default)
       {'keymatched': [list of records matched to key]}
   exact = True
       {'keymatched': record matched to key}
   numerics = True, exact = True
       {'keymatched': (record matched to key, [list of numerics])}

Note: No exception will be raised if partial keys are found or even if no key is found. This is too allow other passed keys to be found.

getrec(tpl, keytpl=None, datafile=None, key=None)

source code 

Convenience function to get a BBPyString record from a BBx data file.

Parameters:
  • tpl - The filename of a BBx Data TPL (template) file.
  • keytpl - The filename of a key TPL file
  • datafile - A BBx Data file from which to retrieve data.
  • key - The key of the record to retrieve and apply the template to.
Returns:
BBPyString of the data in datarec with the templates in tpl and keytpl (if supplied) applied.

Deprecated: Use getRecords instead.
