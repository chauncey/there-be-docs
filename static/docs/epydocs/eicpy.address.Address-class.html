addressClean(self)

source code 

Reformats a street address line to a standard format used by Equity.

  • remove leading # from address
  • abbreviate N W E S NW NE SW SE
  • abbreviate street suffix (Place, Avenue, etc.)
  • Skip direction abbrev when two directions are together (separated by a space). This would take care of 319 WEST SOUTH, but still abbreviate SOUTHWEST FIRST ST.
  • Assume a street suffix (ST, AVE, RD, etc.) or nothing would always follow a street with a direction name. Skip directions followed immediately by a suffix.

removeCityStZip(self)

source code  

Scrub any combination of two or three from this list: City, ST, Zip.
In other words, if the agent typed any of the following at the end of the address line, remove it:
    CITY ST ZIP
    CITY ZIP ST
    ST CITY ZIP
    ST ZIP CITY
    ZIP CITY ST
    ZIP ST CITY
    CITY ST
    ST CITY
    CITY ZIP
    ZIP CITY
    ST ZIP
    ZIP ST
We check the city and state of the zip code entered and use that as the string to match against.
We only check for the 2-character state abbreviation; we do not attempt to find all variations of
abbreviations for the states.

cleanAddrType(self, strToClean, type)

source code 

Input an address string to clean and what aspect of the address that you want cleaned (type). The type value must exist in the AddressMaps table of the addressclean.db database Current type values are: "suffix" and "format" (cannot choose "direction" as a type to clean at this time)
