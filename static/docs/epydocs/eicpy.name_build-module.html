Strips unwanted characters out of a string.

  • remove leading and trailing spaces from string
  • replace multiple spaces with one space
  • keep "-"
  • keep spaces numbers letters
  • convert letters to upper case
  • remove all other characters (including "." and ",")

buildShortName(nameToBuild, strLength, First1, Mid1, Last1, First2, Mid2, Last2)

source code 

For policy with two individuals with same last name, make the compound name fit into the maximum character length following this algorithm

If full name > maximum length, do these steps until full name is maximum length or less:

  1. If the last names are the same, Replace " &/OR " with " & "
  2. If the last names are different, Replace " &/OR " with " / "
  3. Truncate Middle2 to just the initial, M2
  4. Truncate Middle1 to just the initial, M1
  5. Eliminate M2
  6. Eliminate M1
  7. Truncate one letter from First2 until there's one letter left
  8. If there is only one letter left in First2, Truncate one letter from First1 until there's one letter left
  9. For same last name:
    1. Remove "& F2"
    2. Return first "maximum length" characters of name
  10. For different last name:
    1. Replace " / " with "/"
    2. Truncate one letter at a time until length reaches maximum length
