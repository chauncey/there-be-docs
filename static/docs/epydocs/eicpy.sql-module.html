ezFactory is a sqlite3 row_factory for ezRows.

>>> class cur():
...    def __init__(self, s):
...     self.description = []
...     for x in s:
...        self.description.append((x,0))
...
>>> a = ezFactory(cur(["testdesc1","testdesc2","testdesc3"]),[1,2,3])
>>> a.get('testdesc3')
3
>>> a.get('non-existantfield','test')
'test'
>>> a.testdesc3
