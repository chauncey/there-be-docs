zipAllCityStates(zipcode)

source code 

Lookup the city and state associated with a 5 or 9-digit zipcode. 5 digit will return a list of matches.

>>> from eicpy.misc import zipAllCityStates
>>> zipAllCityStates('100010000')
[(u'New York', u'NY')]
>>> zipAllCityStates('74334')
[(u'Miami', u'OK'), (u'Lone Grove', u'OK')]
Parameters:
  • zipcode - 5 or 9-digit zip code
