classifyLocations(self)

source code  
Figure out which location belongs where

From Linda:

1. ITC rate zip may not match passed Vehicle Zip.
- Rate using the Vehicle Zip that ITC passes to us.

2. Zip codes vary for more than one vehicle on a policy.
- 1st Choice: If any vehicle uses the general street address zip code
            THEN use that zip code for all veh
            EXCEPT the 1st alternate zip code will be a garaging zip for that veh.
- 2nd Choice: If more than one vehicle use the same zip code
            THEN use the most common veh zip code for all veh
            EXCEPT the 1st alternate zip code will be a garaging zip for that veh.
- 3rd Choice: If all zip codes are unique or no one zip code is more common
            THEN use the first veh zip code for all veh
            EXCEPT the 1st alternate zip code will be a garaging zip for that veh.
