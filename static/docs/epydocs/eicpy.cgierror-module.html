enable(display=False, logdir=None, context=5, format='html', dev=None, emailonly=False)

source code 

Install an exception handler that reformats tracebacks and notifies IT of the problem.

Parameters:
  • display - if true, tracebacks will be included in the error html page
  • logdir - ignored; retained for cgitb compatibility
  • context - number of lines of source code to show for each stack frame
  • format - 'text' or 'html' controlling the output format
  • dev - list of email addresses to send debug reports to, overrides defaults
  • display - Optional Set to false to suppress sending the traceback to the browser.
  • emailonly - Optional Set to false to suppress sending any content to the browser. Only reports errors via email, does not display an error page.
  • logdir - set a directory for traceback logs.
